Chờ chạy thuật toán...
Parse minterms & don't cares → binary strings
Group implicants by number of 1-bits
FOR each round:
FOR each adjacent group pair (i, i+1):
FOR each pair (impA, impB):
IF differ by exactly 1 bit → merge
Mark impA, impB as usedInMerge = true
Add merged implicant (avoid duplicates)
Collect unused implicants → Prime Implicants
Build PI Chart (minterms only, no don't cares)
Find Essential PIs (columns with single tick)
Apply Petrick's Method for remaining minterms
Generate Boolean expression
